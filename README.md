LinguaConnect AI 🚀

LinguaConnect is a real-time multilingual chat application with built-in AI support.

🔥 Features

- 💬 WhatsApp-style real-time chat UI
- 🌐 Automatic language translation (Telugu ↔ Hindi ↔ English etc.)
- - 🚨 SOS Emergency feature
- 📊 Status update feature (like WhatsApp status)
- 🤖 AI Chat Assistant
- 📷 Image & document support
- 🔒 Secure messaging

🛠 Tech Stack

- React + TypeScript
- Supabase (Backend & Auth)
- AI APIs

📱 Live Demo

👉 https://linguaconnect-ai-c98a5fb4.vercel.app

💡 About Project

This app allows users to chat in their own language while the system automatically translates messages for the other user.

Example:
User A → Telugu
User B → Hindi
👉 Both see messages in their own language
